記帳本進化版

Ledger v2 — 全雲端記帳網頁小專案

這是什麼

一個簡單但完整的記帳網頁：記帳、篩選、月份切換、收支統計、圓餅圖。 重點不是功能多，是「從無到有上線」的完整流程 —— 程式碼、雲端資料庫、自動部署三者串起來。

打開實際網頁 →

第一次點開若休眠中，等約1分鐘即可，不是壞了。

技術組合

FlaskNeon (Postgres)RenderGitHub
  • 資料庫：Neon 雲端 Postgres（免費層，5分鐘閒置自動休息，喚醒快）
  • 部署：Render 常駐後端（免費層，15分鐘無流量休眠，喚醒約1分鐘）
  • 更新流程：改程式碼 → commit/push GitHub → Render 自動重新部署

目前功能

  • 新增／修改／刪除一筆帳
  • 依分類篩選、手動新增分類
  • 月份切換看歷史紀錄
  • 收入／支出／淨額分開顯示
  • 分類支出圓餅圖（隨資料即時變化）
