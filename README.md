# 記帳本進化版（ledger-v2）

## 這是什麼
一個簡單的記帳網頁：記一筆帳（金額、分類、日期、備註），看列表，看本月合計，刪除錯記的一筆。
打開網址就能用，不需要自己的電腦開機——資料庫和網站都放在雲端。

## 用的技術（白話版）
- **網站程式**：Flask（Python 寫網頁的工具）
- **資料庫**：Neon（雲端版的 Postgres 資料庫，免費，5分鐘沒人用會自動休息，下次點開自動醒來，約1~2秒感覺不出來）
- **上線的地方**：Render（免費幫你把網站放到網路上；15分鐘沒人用會休眠，之後第一次點進去可能要等約1分鐘）

## 陽春版功能（目前只有這些）
- 新增一筆帳
- 看列表（最新的在上面）
- 刪除一筆
- 首頁看本月合計

之後可以加：分類統計圖表、月報表、修改帳目、匯出 Excel。

## 更新版本怎麼運作
每次 Claude Code 幫你改完功能，會：
1. 在 `CHANGELOG.md` 最上面加一行，白話說明這次改了什麼
2. `git commit` + `git push` 送到 GitHub
3. Render 看到 GitHub 更新，自動重新部署，你什麼都不用做，等它跑完（通常1~2分鐘）再重新整理網址就是新版本

你不需要懂這些技術細節，只需要打開網址看新功能就好。

## 環境變數
- `DATABASE_URL`：Neon 給的連線字串，設在 Render 的環境變數裡，不要寫進程式碼或 commit 進 git。
